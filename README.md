1. Project Overview
   The Currency Converter is a simple and responsive web application that allows users
   to convert an amount from one currency to another using real-time exchange rates.
   The application fetches live currency data from a public exchange rate API and performs
   accurate currency conversion instantly.



2. Technologies Used

   a-- HTML5
   b-- CSS3
   c-- JavaScript (ES6)
   d-- Exchange Rate API
   e-- Git & GitHub
